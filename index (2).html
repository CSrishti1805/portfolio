System architecture

Designed and built a multi-provider LLM router (Gemini → Groq → OpenAI → Claude) with exponential backoff, provider-specific error classification, and automatic fallback — so the system degrades gracefully under rate limits or auth failures.

Built a FastAPI orchestrator with an agentic loop: the model plans tool calls, executes them concurrently via asyncio.gather, processes results through a context pipeline (relevance filter, dedup, token budget), then synthesises a structured briefing.

Designed a PostgreSQL schema for persistent run history and conversation threads, enabling cross-run trend comparison — the system automatically diffs current findings against prior runs to surface what changed.

Built a React + TypeScript frontend with a chat-thread model (like Claude/ChatGPT), conversation management (rename, pin, share, export to PDF/DOCX/Markdown), and a briefing renderer that parses the agent's prose output into structured sections.

Wrote the system prompt with a STEP 0 triage gate that classifies queries before touching any tools — greetings and out-of-scope questions return in one LLM call with zero tool budget spent.
